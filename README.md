# Академия Дуплея — Платформа самостоятельного обучения

Авторская образовательная платформа Дуплея Максима Игоревича. Содержит 6 курсов (30 уроков) по бизнесу, маркетингу, финансам, менеджменту, ИИ и личной эффективности с геймификацией, квизами, достижениями и сертификатами.

## Технологии

- **Next.js 16** (App Router) + **TypeScript 5**
- **Tailwind CSS 4** + **shadcn/ui** (New York style)
- **Zustand** + persist middleware (хранение прогресса в localStorage)
- **Framer Motion** (анимации)
- **Recharts** (графики прогресса)
- **Lucide icons**, **next-themes** (светлая/тёмная тема)
- **Prisma ORM** (готовая конфигурация, SQLite)

## Запуск

```bash
# Установка зависимостей
bun install

# Запуск dev-сервера
bun run dev

# Открыть в браузере
http://localhost:3000
```

## Возможности

### 📚 Обучение
- **6 авторских курсов, 30 уроков** — теория, кейсы, практические задания
- **Структурированный контент** — введение, ключевые тезисы, теория, задание, ресурсы
- **Поиск и фильтр** по категориям в каталоге

### 🎯 Геймификация
- **90 квиз-вопросов** — по 3 на каждый урок с объяснениями
- **16 достижений** — за прогресс, регулярность, глубину обучения
- **Сертификаты** — генерация красивого сертификата при завершении курса (с возможностью печати в PDF)

### 📊 Аналитика
- **Тепловая карта активности** — 26 недель в стиле GitHub
- **Стрик (streak)** — счётчик последовательных дней обучения
- **Графики прогресса** — Pie chart по категориям, Bar chart по курсам
- **Статистика за 30 дней** — уроки, квизы, заметки, активные дни

### 📝 Личные данные
- **Заметки** — создание, редактирование, удаление в контексте урока
- **Закладки** на курсы и уроки
- **Дашборд** со всей статистикой и последними достижениями
- **Хранение локально** в браузере (localStorage), без сервера

### 🎨 Дизайн
- **Оранжево-янтарная палитра** в OKLCH
- **Светлая/тёмная тема** + полная адаптивность (мобильные/десктоп)
- **Framer Motion** анимации переходов

## Курсы

1. **Основы MBA** (6 уроков, 24 ч) — стратегия, финансы, маркетинг, операции, лидерство
2. **Стратегический маркетинг** (5 уроков, 18 ч) — юнит-экономика, воронки, SEO, бренд
3. **Финансовый менеджмент** (5 уроков, 20 ч) — P&L, моделирование, DCF, оборотный капитал
4. **Управление проектами** (5 уроков, 16 ч) — Agile, Scrum, Kanban, критическая цепь
5. **ИИ для бизнеса** (5 уроков, 14 ч) — LLM, промпт-инжиниринг, RAG, автоматизация
6. **Лидерство и личная эффективность** (5 уроков, 15 ч) — глубокая работа, EQ, decision-making

## Достижения (16 шт)

- **Первый шаг** — завершите первый урок
- **Знаток** — пройдите первый квиз
- **Летописец** — создайте первую заметку
- **В темпе** — завершите 5 уроков
- **Дисциплинированный** — завершите 10 уроков
- **Выпускник** — завершите полностью первый курс
- **Эрудит** — завершите 3 курса
- **Постоянство** — 3 дня подряд
- **Неделя огня** — 7 дней подряд
- **Месяц дисциплины** — 30 дней подряд
- **Конспектирующий** — 10 заметок
- **Экзаменатор** — 10 квизов
- **Коллекционер** — 5 закладок
- **Час силы** — 60 минут обучения
- **Серьёзный подход** — 10 часов обучения
- **Магистр** — завершите все 6 курсов
